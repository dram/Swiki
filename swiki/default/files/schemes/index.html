Color Schemes

Color Schemes can be used to change the look of your Swiki. They don't alter functionality.

Recommend Schemes

There are two general categories of color schemes: those that have the buttons running across the top and those that have the buttons running along the left-hand side. Of the former, the best color schemes (for simplicity and usability) are Poor (minimal icons and simple text) and Blue (Windows 3.1 like icons with text underneath). Of the latter, the best color schemes (for usability) are Aqua and Dock; both are inspired by the Aqua look of MacOS X and feature javascript roll-overs and text below icons.

Other Color Schemes

  • Alien: javascript roll-overs, X-files[esque]
  • Modern Orange: orange and black, no aliasing, boxes around page and book functions
  • Project: made to work with the project Swiki
  • Simple: plain text buttons, easily adaptable
  • Text: plain text (no images)
